Acerca de SneakerComparEsp

Sobre la web

Bienvenid@ a SneakerComparEsp, el comparador definitivo para encontrar los mejores precios para comprar tus sneakers. Esta web es un proyecto del Grado Superior de Formación Profesional de Desarrollo de Aplicaciones Web (DAW), impartido en el FPD de La Rioja, donde muestro las habilidades adquiridas durante el curso, las prácticas y mis años de experiencia en la empresa privada. Además, muestro un poco de mis aficiones ya que es una temática que me encanta sobre un mundo tan bonito como es, para mí, el de las sneakers.

Sobre mí

Mi nombre es Ricardo Calvo Lázaro, soy estudiante de Desarrollo de Aplicaciones Web (DAW), graduado en Sistemas Microinformáticos y Redes (SMR) y Desarrollo de Aplicaciones Multiplataforma (DAM). Soy programador con tres años de experiencia laboral en empresa privada y apasionado de las sneakers.

He trabajado como programador Front-end con Ionic (framework con base de TypeScript para aplicaciones móviles) y Angular (el framework con misma base pero orientado a aplicaciones web) realizando aplicaciones de control laboral, recursos humanos y proyectos a medida para empresas.

Espero que quienes visitéis esta web os guste el estilo un poco más informal que le he intentado dar para salirme de los modelos convencionales de páginas web que puedes encontrar a lo largo y ancho de Internet, apoyéis este trabajo y os contagie un poco mi pasión por las sneakers.

Objetivo del proyecto

He creado esta plataforma como parte de mi Trabajo de Fin de Grado (TFG) del Grado Superior de Formación Profesional de Desarrollo de Aplicaciones Web. El objetivo es mostrar una pequeña parte de mis habilidades de programador Front-end y realizar una Web sobre un tema que me apasiona, algo fuera de lo habitual, para tener una motivación extra a la hora de realizarla.

Además, elegí este proyecto porque, como aficionado del mundo SneakerHead, vi una oportunidad de que la gente obtenga sus zapatillas favoritas al mejor precio con un par de clicks ya que, en este mundillo, encontrar tu modelo favorito se puede volver muy tedioso.
